# Pulsa-Syariah
Pulsa Syariah merupakan aplikasi web marketplace sederhana yang menyediakan layanan pembelian pulsa dan paket data dengan konsep transaksi yang mudah, transparan, dan sesuai dengan prinsip syariah. Aplikasi ini dikembangkan menggunakan framework Flask (Python)  HTML, CSS, JavaScript untuk membangun antarmuka yang responsif.
